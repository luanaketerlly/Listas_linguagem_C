# Listas_linguagem_C
Estruturas de listas

Neste repositório irei colocar em prática os conceitos para criar, editar e excluir itens de listas e ordenação na linguagem C.

Meu objetivo é praticar e fixar os conteúdos que envolvem a matéria de Estrutura de Dados, abrodada no segundo período do curso de Análise e desenvolvimento de sistemas.
